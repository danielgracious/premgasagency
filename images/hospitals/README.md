Until a file exists, that chip shows text initials instead — nothing breaks.
IMPORTANT: only use a hospital's actual logo file with their permission as your
supply partner. Ask their marketing/procurement contact for the official logo
file (usually a transparent PNG) — do not screenshot it off their website.
Recommended: transparent PNG, roughly 200x200px.
